# 헤더 테스트 체크리스트

## 🧪 테스트 환경 설정

### 1. 서버 실행
```bash
npm start
```

### 2. 테스트 페이지 접속
- 메인: `http://localhost:3000/`
- 헤더 테스트: `http://localhost:3000/play/header-test`
- 로그인: `http://localhost:3000/login`

---

## ✅ 비로그인 상태 테스트

### 체크리스트
- [ ] 메인 페이지 접속
- [ ] 로고 표시 확인
- [ ] "로그인" 버튼 표시
- [ ] "회원가입" 버튼 표시
- [ ] "전체 공고 목록" 카테고리만 표시
- [ ] "구직자 탐색" 카테고리 **없음**
- [ ] 로그인 버튼 클릭 → `/login`으로 이동
- [ ] 회원가입 버튼 클릭 → `/signup`으로 이동
- [ ] 로고 클릭 → 메인으로 이동

### 예상 화면
```
┌──────────────────────────────────────────────┐
│ [로고]                  [로그인] [회원가입]    │
├──────────────────────────────────────────────┤
│ 전체 공고 목록                                │
└──────────────────────────────────────────────┘
```

---

## ✅ 개인 회원 로그인 테스트 (로컬)

### 로그인
1. `/login` 페이지 접속
2. `test1@test.com` / `비밀번호` 입력
3. 로그인 버튼 클릭

### 체크리스트
- [ ] 로그인 성공 후 메인 페이지로 리다이렉트
- [ ] "OOO님 환영합니다" 메시지 표시
- [ ] 프로필 정보 표시
  - [ ] 이름 표시
  - [ ] "지원한 기업: X" 표시
  - [ ] 기본 프로필 이미지 표시
  - [ ] 알림 배지 표시 (있으면)
- [ ] 프로필 드롭다운 메뉴 표시
  - [ ] 프로필 관리
  - [ ] 내가 지원한 공고 관리
  - [ ] 로그 아웃 (빨간색)
- [ ] "전체 공고 목록" 카테고리만 표시
- [ ] "구직자 탐색" 카테고리 **없음**

### 드롭다운 메뉴 테스트
- [ ] 프로필 아바타 클릭 → 메뉴 열림
- [ ] 외부 클릭 → 메뉴 닫힘
- [ ] ESC 키 → 메뉴 닫힘
- [ ] "로그 아웃" 클릭 → 로그아웃 + 메인 페이지

### 예상 화면
```
┌──────────────────────────────────────────────────────────┐
│ [로고]   홍길동님 환영합니다 [👤 홍길동] [📋 프로필메뉴] │
│                              지원한 기업: 3               │
├──────────────────────────────────────────────────────────┤
│ 전체 공고 목록                                            │
└──────────────────────────────────────────────────────────┘
```

- [ ] 카카오 닉네임 표시
- [ ] "지원한 기업: X" 표시
- [ ] 알림 배지 표시
- [ ] 나머지는 로컬 로그인과 동일

### Mock 테스트 (개발 환경)
1. `/play/header-test` 접속
2. "개인 회원 로그인 (카카오)" 버튼 클릭
3. 카카오 이미지 확인

## ✅ 기업 회원 로그인 테스트

### 로그인
1. `/login/company` 페이지 접속
2. 기업 계정 입력
3. 로그인 버튼 클릭

### 체크리스트
- [ ] "OOO기업님 환영합니다" 메시지 표시
- [ ] 프로필 정보 표시
  - [ ] 기업명 표시
  - [ ] "채용 공고: X개" 표시
  - [ ] 프로필 이미지 표시
  - [ ] 알림 배지 표시 (지원자 알림)
- [ ] 프로필 드롭다운 메뉴 표시
  - [ ] 프로필 관리
  - [ ] 공고 관리 ⭐
  - [ ] 지원자 관리 ⭐
  - [ ] 로그 아웃 (빨간색)
- [ ] "전체 공고 목록" 카테고리 표시
- [ ] "구직자 탐색" 카테고리 표시 ⭐

### 예상 화면
```
┌──────────────────────────────────────────────────────────┐
│ [로고]   ABC기업님 환영합니다 [🏢 ABC기업] [📋 메뉴]      │
│                              채용 공고: 8개               │
├──────────────────────────────────────────────────────────┤
│ 전체 공고 목록    구직자 탐색 ⭐                          │
└──────────────────────────────────────────────────────────┘
```
